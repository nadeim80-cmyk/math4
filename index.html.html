🔢 What is Place Value?
The value of each digit in a number depends on its position (place).
We group digits into Periods — each period has 3 digits: Ones, Tens, Hundreds.
📌 There are 4 Periods you need to know:
■ ONES  |  ■ THOUSANDS  |  ■ MILLIONS  |  ■ MILLIARDS (Billions)

💡 Key Facts to Remember
📌 Number of Digits in Each Period's Smallest Number:
Smallest 6-digit number = 100,000
Smallest 7-digit = 1,000,000 (1 Million)
Smallest 10-digit = 1,000,000,000 (1 Milliard)
Largest 6-digit = 999,999
🎯 Million = smallest number formed from 7 digits
🎯 Milliard = smallest number formed from 10 digits
🎯 Each place value is 10 times the one to its right!
✨ Solved Examples
EXAMPLE 1 — From the Book
In 1,542,345,678 — what digit is in the Ten Millions place?
→ Count from right: Ones, Tens, Hundreds, Thousands, Ten-Thousands, Hundred-Thousands, Millions, Ten Millions
→ The digit in Ten Millions place = 4
EXAMPLE 2 — Place Value of 7 in 7,213,455,686
The number has 10 digits → the first digit is in the Milliards place.
→ Place value of 7 = Milliards
EXAMPLE 3 — Exam Style (Luxor 24)
The digit in Ten Thousands place in 1,351,278 is:
→ Ones=8, Tens=7, Hundreds=2, Thousands=1, Ten Thousands=5
→ Answer = 5
